# singbox-manager 轻量化开发路线图

## 当前阶段

当前项目已经进入：

**0.x 可发行可用阶段**

定位：

- 轻量
- 高性能
- CLI + TUI
- 单机 / 小规模管理优先
- 不做 Web

当前适合：

- 自用
- 内部工具
- 单机 sing-box 管理
- 小规模发行版本

---

## 当前已完成的核心能力

### 已完成

- 用户管理闭环
  - 添加 / 删除 / 启禁 / 重置 / 套餐更新 / 手动流量调整
  - 用户凭据生成
  - 用户变更自动同步 sing-box 配置
  - 自动同步 `experimental.v2ray_api.stats.users`
  - 配置校验 + 运行中自动 reload

- 流量统计闭环
  - gRPC 查询
  - 用户流量增量统计
  - 写入数据库
  - 自动控制（月重置 / 到期禁用 / 超额禁用）

- 服务管理闭环
  - `sb check`
  - `sb start`
  - `sb stop`
  - `sb reload`
  - `sb status`

- 节点最小部署闭环
  - `sb add-node`
  - 写入 sing-box `inbounds`
  - 保存配置
  - 校验配置
  - 自动 reload

- 订阅导出
  - 明文导出
  - Base64 订阅导出
  - 常见协议基础链接格式

- TUI 主面板
  - 浏览用户 / 节点 / 日志 / 状态
  - 启禁用户
  - 重置流量
  - 刷新状态
  - 配置校验
  - 日志页显示最近一次订阅导出

## 当前项目边界

继续保持：

- 不做 Web
- 不做复杂权限系统
- 不做平台化后台
- 只围绕 sing-box 部署、用户、统计、订阅展开
